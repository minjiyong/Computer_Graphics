<폴리곤 슬라이스> - 2021182012 민지용

기본 구현 내용
- 화면의 좌측, 우측 가장자리 상단 부분(y 좌표 0.41f ~ 0.80f)의 
  임의의 위치에서 도형이 화면의 반대편으로 날아온다.
- 날아오는 경로는 포물선으로 설정했고, 중력가속도를 적용했다.

- 마우스를 클릭, 드래그 하고 떼면 도형이 슬라이스 된다.
- 도형을 관통(선과 도형의 교점이 2개) 해야만 슬라이스 되고, 
  교점이 1개라면 슬라이스 되지 않는다.
- 슬라이스 된 도형은 좌우로 조금씩 갈라지며 아래로 떨어진다.

- 화면 아래쪽에 바구니가 좌우로 움직인다. 
- 바구니에 슬라이스 된 도형의 vertex중 가장 y값이 작은 녀석이 충돌하면 바구니 위에 놓여진다.

- 키보드 명령어 w/W 를 사용해 도형을 LINE/FILL 로 출력한다. (기본: FILL)
- a/A 를 사용해 경로를 출력/출력X  한다. (기본: 출력X)
  경로는 난잡하게 보이기 때문에 도형을 슬라이스 함과 동시에 삭제한다.
- +/- 를 사용해 날아오는 속도와 중력 가속도를 가속/감속한다.
  최대 속도는 Speed Level 9, 최저 속도는 0 이다. (기본: 2)
- q 를 눌러 프로그램 종료
- s 를 눌러 게임 재시작


보너스 구현 내용
- 자유 슬라이스, 슬라이스 될 때 그린 선분대로 도형이 나누어져 떨어진다.
- 한 번 슬라이스 한 도형을 무한히 슬라이스 할 수 있다.
- 도형이 회전하면서 날아온다.


추가 구현 내용
- 슬라이스 가 일어나면 이펙트가 발생함.
  이펙트는 3~8각형 도형이 작게 랜덤 방향으로 날아가다 사라지게 함.
- 도형을 기본 삼각형과 사각형 외에도 3~8각형 이 랜덤으로 나오게 함.
- 도형이 범위를 벗어나면 삭제한다. 이 때 경로도 같이 삭제한다.
- 바구니 윗면에 정확히 충돌해야만 바구니 위에 놓여진다. 
- 우클릭을 하면 해당 위치에 도형이 생성됨 (몇각형일지, 어느 방향일지는 랜덤)
